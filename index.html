FALLSTRUKTUR Scene Editor

This website is a very janky and unpolished scene editor. Please let me know if you encounter any bugs in usage at the GitHub page.

Thanks to:

  • Intstruck_dream for creating the first variable modified levels
  • Safariminer for initially working on Openstruktur
  • proalfacracky & Koopa for pioneering the public knowledge of modifying the levels

Due to the way this tool is made it's required to keep the modifications made to your scene. At the bottom, there is a button to save your configuration. I recommend as best practice to keep this file in the folder of your world.scene file.

This uses the same files as the above, however instead of the expected input, it sets the expected output. Use it to load shared presets. You do not need to use this to use presets, however it's an easy way to do so.
